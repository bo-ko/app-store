What’s New In python-apt 1.1¶

This release is built against APT 1.1

Highlights¶

  • Code that previously raised SystemError now raises apt_pkg.Error.

Removed¶

  • The class apt_pkg.IndexRecords has been removed, as it was removed in APT 1.1
  • apt_pkg.Dependency.smart_target_pkg has been removed.

Added¶

  • The class apt_pkg.HashStringList has been added.
  • The class apt_pkg.Error and an alias apt_inst.Error has been added.

Deprecated¶

  • apt_pkg.PackageRecords.md5_hash, apt_pkg.PackageRecords.sha1_hash, and apt_pkg.PackageRecords.sha256_hash are replaced by apt_pkg.PackageRecords.hashes.
